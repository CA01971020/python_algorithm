# 問題: 文字の出現回数カウント（初級）

英小文字のみからなる文字列 s が与えられます。この文字列に含まれる各文字が何回登場したかを数えて、アルファベット順に表示してください。

関数名： count_characters(s: str) -> None

## 🔧 入力例
```
count_characters("banana")
```

## 🧪 出力例
```
a: 3
b: 1
n: 2
```

## 条件
- 出力の順序は**アルファベット順（a〜z）**とする。
- 出現しなかった文字は表示しない。
- 出力形式は "文字: 数"（例: a: 3）としてください。

## ヒント
- collections.Counter を使うと便利ですが、まずは辞書（dict）を使って自分で実装してみると力になります。
- sorted() を使えばキーをアルファベット順に並べることができます。
